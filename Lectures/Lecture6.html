Lecture 6: Forms and Multimedia

Web Forms

  • Allow users to input and submit data.
  • Forms consist of controls (widgets) like text boxes, checkboxes, and buttons.

Form Controls and Widgets

  • Controls: text boxes, radio buttons, checkboxes, text areas, selection lists
  • Widgets: sliders, spin boxes, date pickers, color pickers

Form Tag & Attributes

<form action="/action_page.php" method="post" target="_blank">
    ...form elements...
</form>

Common attributes: action, method, target, name, id

Input Types and Examples

<input type="text" placeholder="Your name">
<input type="password">
<input type="radio" name="gender" value="male" checked> Male
<input type="checkbox" name="vehicle" value="Bike"> I have a bike
<input type="submit" value="Submit">
<input type="reset" value="Reset">
<input type="button" value="Click Me!" onclick="alert('Hello')">

Label Example

<label for="fname">First Name:</label>
<input type="text" id="fname" name="fname">

Advanced Input Types (HTML5)

  • email, url, tel
  • date, datetime-local, month, week, time
  • color, range, search

Color Picker Example

<input type="color" id="favcolor" name="favcolor">

Input Attributes

  • value, readonly, disabled, size, maxlength
  • HTML5: autocomplete, autofocus, required, placeholder, pattern

Form with Autocomplete

<form autocomplete="on">
    <input type="text" name="fname">
    <input type="email" name="email" autocomplete="off">
</form>

Select Element

<select name="movies">
    <option value="Ten" selected>Ten Things I Hate About You</option>
    <option value="Fifty">Fifty First Dates</option>
</select>

Textarea

<textarea id="message" name="message" rows="4" cols="30">
The cat was playing in the garden.
</textarea>

Fieldset and Legend

<fieldset>
    <legend>Personal Info</legend>
    <input type="text" name="fname" value="Nicole">
</fieldset>

Creating Effective Forms

  • Adjust field sizes for input clarity
  • Group fields logically with <fieldset>
  • Use input types wisely (radio, checkbox, select)
  • Use labels for accessibility

Multimedia Elements

Audio

<audio controls>
    <source src="sound.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

Video

<video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

Autoplay Video

<video width="320" height="240" autoplay>
    <source src="movie.mp4" type="video/mp4">
</video>

Embedding Plug-ins

Using <object>

<object width="400" height="200" data="file.swf"></object>

Using <embed>

<embed width="400" height="50" src="file.swf">
