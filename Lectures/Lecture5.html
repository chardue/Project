Lecture 5: CSS3

Introduction to CSS3

  • Originated from SGML in the 1980s.
  • Aims to separate structure from presentation.
  • CSS Level 1 was released in 1996 by W3C, CSS Level 2 in 1998, and CSS3 is modular.
  • Over 50 modules published by the W3Câ€™s CSS Working Group.
  • CSS3 is backward compatible with CSS 2.1.

CSS Solved a Big Problem

  • HTML was not meant to include formatting tags.
  • CSS removes styling from HTML, simplifying large website management.

CSS Syntax and Selectors

  • A CSS rule-set includes a selector and a declaration block.
  • Selectors target elements; declarations define style.
  • Declarations end with semicolons and are inside curly braces.

Element Selector

p {
        text-align: center;
        color: red;
        }

ID Selector

        #para1 {
        text-align: center;
        color: red;
        }

Class Selector

.center {
    text-align: center;
    color: red;
}

Grouping Selectors

h1, h2, p {
    text-align: center;
    color: red;
}

CSS Comments

/* This is a comment */
/* This is
a multi-line
comment */

Three Ways to Insert CSS

  • External Style Sheet (linked in <head>)
    • <link rel="stylesheet" href="mystyle.css">
  • Internal Style Sheet
    • <style>
body {
    background-color: linen;
}
</style>
  • Inline Style
    • <h1 style="color:blue;margin-left:30px;">Heading</h1>

Cascading Order (Priority)

  1. Inline style
  2. Internal style sheet
  3. External style sheet
  4. User-defined styles
  5. Browser defaults

CSS Colors

  • Background Color: <h1 style="background-color:DodgerBlue;">
  • Text Color: <h1 style="color:Tomato;">
  • Border Color: <h1 style="border:2px solid DodgerBlue;">

Color Values

  • RGB: rgb(255, 0, 0)
  • HEX: #ff0000
  • HSL: hsl(0, 100%, 50%)
  • RGBA: rgba(255, 0, 0, 0.5)
  • HSLA: hsla(0, 100%, 50%, 0.5)

CSS Backgrounds

body {
    background-image: url("img_tree.png");
    background-repeat: no-repeat;
    background-position: right top;
    background-attachment: fixed;
}

Shorthand:

body {
    background: #ffffff url("img_tree.png") no-repeat right top;
}

CSS Borders

p.dotted {border-style: dotted;}
p.dashed {border-style: dashed;}
p.solid {border-style: solid;}
p.double {border-style: double;}

Border Width and Color

p {
    border-width: thick;
    border-color: red;
}

Individual Sides

p {
    border-top-style: dotted;
    border-right-style: solid;
    border-bottom-style: dotted;
    border-left-style: solid;
}

Shorthand Border

p {
    border: 5px solid red;
}

Left and Bottom Borders

p {
    border-left: 6px solid red;
    background-color: lightgrey;
}

p {
    border-bottom: 6px solid red;
    background-color: lightgrey;
}

Rounded Borders

p {
    border: 2px solid red;
    border-radius: 5px;
}

CSS Height and Width

  • Default is auto
  • Can use px, cm, %, etc.

Max Width

div {
    max-width: 500px;
}
