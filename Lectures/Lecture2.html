What is HTML?

HTML (Hypertext Markup Language) is a markup language that structures web pages. HTML5 is the latest version, aiming to:

  • Support all existing pages
  • Reduce dependency on plugins
  • Improve semantics and error handling

HTML5 Timeline

Versions include HTML2.0, HTML3.2, HTML4.01, XHTML1.0, XHTML2.0, and HTML5.

New Elements in HTML5

Examples: <article>, <nav>, <footer>, <section>, <canvas>

DOCTYPE Declaration

<!DOCTYPE html>

HTML5 Page Skeleton

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="utf-8">
            <title>My First HTML5 Page</title>
            <link rel="stylesheet" href="style.css">
        </head>
        <body>
            <p>HTML5 is fun!</p>
        </body>
        </html>

HTML Concepts

  • Elements: Containers like <h1>Title</h1>
  • Tags: Opening and closing brackets
  • Attributes: Define characteristics (e.g. align="center")
  • Nesting: Never overlap elements improperly

Case Sensitivity & Comments

HTML is not case-sensitive, but lowercase is recommended. Comments use:

<!-- This is a comment -->

Semantic Elements in HTML5

  • <header>, <footer>, <article>, <section>, <nav>, <aside>

Text Formatting Tags

  • <b>, <strong>, <i>, <em>, <mark>, <u>, <del>, <ins>
  • <sub>, <sup>, <code>, <kbd>, <samp>

Paragraphs & Line Breaks

  • <br>: Line break
  • <p>: Paragraph
  • &nbsp;: Non-breaking space

HTML Entities

  • &lt; = <
  • &gt; = >
  • &amp; = &
  • &copy; = ©

Indentation and Alignment

Use <blockquote> for block indentations. Use align="center", right, or justify in elements like <p>.
