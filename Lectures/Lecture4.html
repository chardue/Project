HTML LISTS

  • Making information more accessible is the single most important quality of HTML.
  • It helps you organize your information into documents readers can quickly understand, scan, and extract, possibly with automated browser agents.
  • It shows specific information quickly and clearly.

Types of Lists

  • Ordered list <ol>
  • Unordered list <ul>
  • Definition list <dl>

Ordered Lists

Lists in which each item is numbered, usually starting at 1. Useful for ordered or hierarchical items such as instructions.

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

With CSS:

    HTML:
<ol>
  <li>Notebook</li>
  <li>Ballpen</li>
  <li>Yellow Paper</li>
</ol>

CSS:
li {
  list-style-type: upper-roman;
}

list-style-type Values

  • decimal – default numbers
  • upper-alpha – uppercase letters
  • lower-alpha – lowercase letters
  • upper-roman – uppercase Roman numerals
  • lower-roman – lowercase Roman numerals

Attributes:

  • type="1" – specify list type
  • start="5" – starting point of the list
<ol type="A" start="5">
  <li>Item</li>
</ol>

Unordered Lists

Used to create bulleted lists.

<ul style="list-style-type:square">
  <li>Notebook</li>
  <li>Ballpen</li>
  <li>Yellow Paper</li>
</ul>

Bullet Types:

  • disc – default bullet
  • circle
  • square
  • none – no bullets

Description Lists

<dl>
  <dt>Notebook</dt>
  <dd>- a small book with blank or ruled pages for writing notes in.</dd>
  <dt>Ballpen</dt>
  <dd>- a pen that dispenses ink over a metal ball at its point</dd>
</dl>

Nested List

<ul>
  <li>Coffee</li>
  <li>Tea
    <ul>
      <li>Black tea</li>
      <li>Green tea</li>
    </ul>
  </li>
  <li>Milk</li>
</ul>

Horizontal List with CSS (Navigation Menu)

Insert to <head>:

<style>
ul {
  list-style-type: none;
  margin: 0;
  padding: 0;
  overflow: hidden;
  background-color: #333333;
}
li {
  float: left;
}
li a {
  display: block;
  color: white;
  text-align: center;
  padding: 16px;
  text-decoration: none;
}
li a:hover {
  background-color: #111111;
}
</style>

Insert to <body>:

<ul>
  <li><a href="#home">Home</a></li>
  <li><a href="#news">News</a></li>
  <li><a href="#contact">Contact</a></li>
  <li><a href="#about">About</a></li>
</ul>

HTML Links / Hyperlinks

Links let users jump from one page to another or to a specific location in the current page.

Types of href Values

  • Absolute URL
  • Relative URL
  • Jump within the current page
<a href="home.html">Back to HOME</a>

Absolute link:

<a href="http://google.com">Google</a>

Relative link:

<a href="template.htm">Template</a>

Internal Link

<a href="#sectionID">Jump to Section</a>

Link Colors (CSS)

  • a:link – unvisited link
  • a:visited – visited link
  • a:hover – mouse hover
  • a:active – clicked link

target Attribute

  • _blank – new tab/window
  • _self – same tab (default)
  • _parent – parent frame
  • _top – top frame
  • framename – named frame

<a href="template.htm" target="_top">HTML5 tutorial!</a>

Images

Best Web Image Formats

  • WebP – replacement for JPEG, PNG, GIF
  • GIF – good for drawings, animations
  • JPEG – best for real photos
  • PNG – combines GIF and JPEG benefits

GIF Features

  • Lossless compression
  • Max 256 colors
  • GIF89a supports transparency, animation
  • Interlacing for progressive loading

JPEG Features

  • Best for photorealistic images
  • Supports thousands of colors

PNG Features

  • Lossless compression
  • Supports alpha transparency and multiple color depths
